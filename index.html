Bienvenue sur la page du projet CalorieBurner

Cette application android a été réalisé dans le cadre d'un projet informatique de 3ème année d'école d'ingénieurs qui s'est déroulée en 2 parties:
1. Une implémentation UML (Unified Modeling Language) de l'ensemble de l'application
2. Un développement sur plateforme android 4.2.2

Qu'est ce que l'on peut faire avec ?

Tout le monde n'est pas sportif mais tout le monde mange, avec CalorieBurner vous allez pouvoir faire les actions suivantes:

  • Composer vos repas.
  • Éliminer les calories consommées en faisant un sport ou une activité physique.
  • Faire du sport.
  • Partager vos résultats sur les réseaux sociaux
    Attention elle ne se substitue pas un diététicien

Si je suis perdu ?

  • Une aide est disponible sur toutes les fenêtres de l'application
